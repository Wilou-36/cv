💳 Application Bancaire

Application web de gestion de comptes avec simulation crypto

🧠 Contexte

Projet réalisé en formation BTS SIO SLAM visant à développer une application web simulant un environnement bancaire.

🎯 Objectif

Permettre la gestion de comptes utilisateurs et intégrer une fonctionnalité de simulation de cryptomonnaies en temps réel.

⚙️ Fonctionnalités

  • Gestion des comptes utilisateurs
  • Transactions (crédit / débit)
  • Simulation de cryptomonnaies

🛠️ Stack technique

  • Mettre à disposition des utilisateurs un service informatique
